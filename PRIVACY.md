# Privacy

- Authentication: Anonymous Firebase Auth; no email or PII collected by default.
- Data stored: partner link doc, shared randomizer state, plans, lists, game, and stats.
- Visibility: Data is visible only to the two linked users per security rules.
- Removal: Clearing app/storage or unlinking removes local access; contact project owner to request remote deletion.
- Environment: You control the Firebase project and its data retention policies.
